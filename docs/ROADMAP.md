# ImmersiAds — Roadmap

## Version 1.0 (Current — MVP)

- [x] User onboarding with language selection
- [x] Advertisement feed with language filtering
- [x] Video playback with ExoPlayer
- [x] Variable playback speed
- [x] Interactive subtitles with word tapping
- [x] Word translation popup
- [x] Save vocabulary to local Room database
- [x] Personal vocabulary list with search
- [x] Learning streaks
- [x] Progress dashboard
- [x] App settings
- [x] GitHub Actions CI pipeline

## Version 1.1 — Enhanced Learning

- [ ] Spaced repetition vocabulary review (SRS)
- [ ] Vocabulary quiz mode (multiple choice, fill-in-the-blank)
- [ ] Full sentence translation (not just words)
- [ ] Vocabulary export (CSV, Anki deck)
- [ ] Offline mode — pre-download advertisements

## Version 1.2 — Social & Gamification

- [ ] User accounts and cloud sync
- [ ] Leaderboards and community streaks
- [ ] Badges and achievements
- [ ] Share vocabulary snippets

## Version 2.0 — Expanded Content

- [ ] Real advertisement API integration
- [ ] Content moderation and safety filtering
- [ ] Multiple difficulty tracks per language
- [ ] User-submitted subtitle corrections
- [ ] Additional immersive media sources (TV show clips, short films)
